    + - Class

    • 一个类只能有一个静态构造函数，

      该函数不能有访问修饰符，不能带参数，也不能被用户直接调用。但他会在以下情况被隐式调用：

      a、当包含静态构造函数的类实例被首次创建时；

      b、当使用包含静态构造函数的类的其他静态变量时；

      结构类型是值类型

      class myClass{

       //personal code;

      }

      默认是internal访问修饰符，可以使用public；

      类的修饰符还可以用abstract或者sealed（相当于final）；

      public class A ：B，IC，ID

      编译器不允许派生类的可访问性高于基类；

      构造函数初始化器

      public a(int i,int j):base(i){}

      base关键字是指在实例化过程中使用指定的基类构造函数；

      public a(int i,int j):this(i){}

      readonly，static,const,

      get,set块

      private int myInt;

      public int MyInt{

      get{

      return myInt;}

      protected set{

      myInt=value;}

      }

      + - System.nullable<T>

      • e.g.

        System.nullable<int>可以简写为int?

        所以定义一个System.nullable<int>的泛型实例的话，可以：

        int? nullint;

        可以使用value来访问nullint的值；

      空接合运算符（注意汉字“接合”）：??

      a??b;

      等于：

      a==null?b:a;

      要创建泛型类型，需要明确它是引用类型还是值类型
